Projects

Resound

Resound is a music-sharing app where musicians can post their work and listen to other artists.

It is a fullstack Soundcloud clone, utilizing ajax requests to allow the user to navigate the app while playing music

I built CRUD methods for songs and comments for user accounts with Rails and created user and song pages with React-Redux.

I also created a responsive audio player form the ground-up that plays, scrubs, and loops through songs.

Pinbrawl Wizard

Pinbrawl Wizard is a JS video game where a plucky penguin battles hordes of enemies in a game of pinball.

The frontend Javascript game was made using only HTML Canvas.

I coded realistic physics collisions and objects from scratch, calculating how the pinballs and enemies collide and bounce off each other.

A drag-and-drop image upload feature is implemented to allow players to create their own enemies.

Dayze

An event-tracking application

I used the MERN stack to handle user authentication, event CRUD, and notification CRUD.

I implemented event timers that create reminders of current tasks on a different page using React-Redux and JS.

I built another feature in the works with Google Calendar API that syncs Dayze app with Google Calendar.
